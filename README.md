## 🛠 Installation

1. Install deps with NPM or Yarn:

```bash
npm install
#or
yarn install
```

2. In the project directory, run the development server:

```bash
npm start
#or
yarn start
```

## 🚀 Build and Run for Production

Generate full static build for production:

```bash
npm run build
#or
yarn build
```
